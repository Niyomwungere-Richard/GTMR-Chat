# **App Name**: ConnectSphere

## Core Features:

- User Authentication: User authentication and profile management, including sign-up, login, and profile editing.
- Real-Time Chat: Real-time chat functionality, enabling one-on-one conversations and group chats.
- Content Sharing: Content sharing feature, allowing users to post updates, images, and links on a personal or group feed.
- Smart Content Suggestions: AI-powered content suggestion tool, intelligently suggesting relevant content based on user interests.
- News Feed: News Feed Aggregator: Pulls together the most relevant updates from friends and followed accounts into a reverse-chronological feed
- Reactions and Comments: A system to enable reactions and commenting on shared content.

## Style Guidelines:

- Primary color: Soft, calming blue (#79A6D2) to foster a sense of connection and trust.
- Background color: Very light blue (#F0F4F8) providing a clean, unobtrusive backdrop for content.
- Accent color: Muted orange (#D29179) to highlight key actions and notifications without overwhelming the user.
- Body and headline font: 'PT Sans', a versatile humanist sans-serif.
- Simple, consistent icons to ensure intuitive navigation.
- Clean, card-based layout to display content clearly.
- Subtle transitions and animations to provide a smooth user experience.
